Crossover Shopping Cart e-commerce Web

All the ways you mean. All we have for you.

This application consists of:

  • Sample pages showing basic nav between Home, About, and Contact
  • Theming using Bootstrap
  • Authentication, if selected, shows how to register and sign in
  • ASP.NET MVC 5 with WebApi & EF 5.X features managed using NuGet

Details on Website

Pages:

  • Home page for shopping cart
  • Login page
  • Registration page
  • Product page to add new product for otherize user. i.e Operator
  • All products page for both registered user & anonymus user to add products
  • View-cart page for both registered user & anonymus user to check the added products and price
  • Continue shopping button to stay in the cart. & Procced to pay will take you to payment details page.
  • Upon entering the details and pressing "PAY" button will show you the successful message for order confirmation.
